Среди перечисленных утверждений выберите верные
1) структура не может служить возвращаемым значением метода
2) структура может применяться в качестве параметра, передаваемого по значению
3) структура не может служить параметром, снабженным модификатором <code>ref</code>
4) структура не может служить параметром, снабженным модификатором <code>out</code>
5) метод не может изменять значения полей структуры, использованной в качестве параметра
