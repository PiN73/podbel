В программе описан алгоритм для работы устройства контролирующего состояние воды. Если температура ниже нуля градусов выводится сообщение "лёд", если температура больше либо равна нулю градусов — сообщение "вода", если же температура выше либо равна ста градусам - сообщение "пар"
<code>int t = 100;					// 1
if (t > 100) Console.Write("пар");		// 2
if (t >= 0 && t < 100) Console.Write("вода");	// 3
if (t < 100) Console.Write("лёд");		// 4
</code>
Укажите, какие изменения могут быть внесены в программу, если известно, что приведенный фрагмент не выводит информацию на экран:
1) В строке 2 заменить знак <code>></code> на <code>==</code>
2) В строке 2 заменить знак <code>></code> на <code>>=</code>
3) В строке 3 заменить операцию <code>&&</code> на <code>||</code>
4) В строке 3 заменить операцию <code><</code> на <code><=</code>
5) В строке 2 заменить знак <code>></code> на <code>==</code>, в строке 3 заменить операцию <code><</code> на <code><=</code>
