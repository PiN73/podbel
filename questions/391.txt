Среди перечисленных утверждений выберите верные:
1) модификатор virtual нельзя использовать вместе с модификатором override
2) по умолчанию все методы в абстрактном классе являются виртуальными
3) виртуальные методы нельзя объявить в неабстрактном классе
4) модификатор virtual нельзя использовать совместно с модификатором public
5) конструктор без параметров, добавляемый в объявление класса компилятором, по умолчанию является виртуальным
