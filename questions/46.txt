В языке С# может быть исполнено автоматическое преобразование типа:
1) из типа <code>char</code> в тип <code>uint</code>;
2) из типа <code>char</code> в тип <code>double</code>;
3) из типа <code>uint</code> в тип <code>char</code>;
4) из типа <code>double</code> в тип <code>char</code>;
5) из типа <code>uint</code> в тип <code>char</code>, если значение переменной типа <code>uint</code> нe превышает 65535.
