Укажите номера всех истинных высказываний из приведенного списка:
1) Метод перегрузки операции всегда является открытым статическим методом включающего его класса, поэтому может наследоваться производным классом
2) При перегрузке бинарной операции по крайней мере один параметр должен иметь тип класса, включающего метод перегрузки операции
3) Для перегрузки префиксной операции <code>++</code> метод перегрузки операции должен быть методом объекта
4) У метода перегрузки операции приведения типов тип параметра может не совпадать с типом включающего его класса
5) Метод перегрузки операции для бинарной операции может иметь два параметра с типом класса, базового по отношению к включающему его классу
