1;3;4;5;

Укажите номера верных высказываний. Верно, что:
1) Исключение как объект создается с помощью выражения с операцией <code>new</code>
2) Оператор <code>throw</code> служит для создания объекта-исключения
3) Из нескольких <code>catch</code> обработчиков одного блока контроля может выполняться только один
4) Все обработчики исключений одного блока контроля должны иметь разные спецификации параметров
5) Оператор посылки исключения в формате <code>throw;</code> может использоваться только в <code>catch</code>-секции
